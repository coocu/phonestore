📄 개인정보 처리방침

본 개인정보 처리방침은 「폰스토어(PhoneStore)」 앱(이하 “본 앱”)에서 제공하는 서비스 이용과 관련하여, 이용자의 개인정보 처리에 관한 사항을 안내하기 위하여 작성되었습니다.

1. 수집하는 개인정보 항목

본 앱은 아래와 같은 항목을 저장하거나 서버로 전송하지 않습니다.

이름, 전화번호, 이메일

기기 고유식별정보(IMEI, UUID 등)

광고 식별자(ADID)

앱 이용 로그 데이터

2. 위치정보 처리에 관한 사항

본 앱은 사용자의 현재 위치를 기반으로 주변 매장 정보를 표시하기 위해
다음 항목을 이용합니다.

📍 수집 항목

기기 위치 정보(GPS / 네트워크 기반 위치)

📌 이용 목적

현재 위치 기준 주변 매장 거리 계산

지도 화면에서 매장 위치 표시

❗ 위치 정보 서버 전송 여부

위치 정보는 사용자 기기에서만 처리

외부 서버로 전송 ❌

저장 ❌

로그 기록 ❌

사용자는 단말기 설정에서 언제든지 위치 권한을 철회할 수 있습니다.

3. 외부 데이터 사용 안내

본 앱은 매장 정보를 제공하기 위해 아래 공개 저장소에서
매장 데이터(JSON)를 불러옵니다.

GitHub 저장소

사용 목적 : 매장 위치 및 연락처 정보 표시

이 과정에서 이용자의 개인정보는 전송되지 않습니다.

4. 권한 사용 안내

본 앱은 다음 권한을 사용합니다.

(선택) 위치 권한

ACCESS_FINE_LOCATION / ACCESS_COARSE_LOCATION

목적 : 주변 매장 안내 및 지도 표시

(필수) 인터넷 권한

INTERNET

목적 : 매장 정보 JSON 불러오기

(선택) 네트워크 상태 권한

ACCESS_NETWORK_STATE

목적 : 네트워크 연결 확인

본 권한은 개인정보 수집과 무관합니다.
